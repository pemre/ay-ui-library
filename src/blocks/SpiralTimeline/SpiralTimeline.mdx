# SpiralTimeline

A D3-powered spiral timeline visualization for React. Each concentric ring represents one calendar year, months are arranged as radial sectors, and data nodes are plotted at their calendar positions.

<Canvas of={SpiralTimelineStories.Default} />

## Usage

```tsx
import { SpiralTimeline } from "ay-ui-library";
import "ay-ui-library/dist/style.css";

const data = [
  {
    date: new Date("2024-03-15"),
    type: "history",
    title: "Spring Equinox",
    content: "The vernal equinox marks the start of spring.",
  },
  {
    date: new Date("2024-07-20"),
    type: "science",
    title: "Moon Landing Anniversary",
    content: "Celebrating the Apollo 11 mission.",
  },
];

function App() {
  return (
    <SpiralTimeline
      data={data}
      config={{ yearsToShow: 3 }}
      locale="en"
      onNodeClick={(node) => console.log("Clicked:", node.title)}
    />
  );
}
```

## Data Format

Each item in the `data` array must conform to the `DataNode` interface:

```ts
interface DataNode {
  date: Date;       // Calendar date for spiral positioning
  type: string;     // Maps to a TypeConfig entry for color/shape
  title: string;    // Shown in tooltip and aria-label
  content: string;  // Shown in tooltip body
  id?: string;      // Optional stable key for data updates
  metadata?: Record<string, unknown>; // Pass-through for extra fields
}
```


The `type` field maps to entries in `config.types` to determine the node's color and shape. If no match is found, the first entry in `types` is used as a fallback.

## Theming

SpiralTimeline uses a three-tier CSS custom property architecture:

1. **Core tokens** — raw design values (`--color-gray-500`, `--space-2`)
2. **Semantic tokens** — theme-aware mappings (`--color-bg-surface`, `--color-text-primary`)
3. **Component tokens** — component-specific overrides (`--spiral-bg`, `--spiral-tooltip-bg`)

The component reads semantic tokens and maps them to `--spiral-*` component tokens. Override any component token to customize the look:

```css
.my-wrapper .spiral-timeline {
  --spiral-bg: #0a0a1a;
  --spiral-primary: hotpink;
  --spiral-tooltip-bg: #1a1a2e;
  --spiral-tooltip-border: #333;
}
```

Theme switching works automatically via the `data-theme` attribute on a parent element:

```html
<div data-theme="dark">
  <SpiralTimeline data={data} />
</div>
```

## Responsive Behavior

The component adapts to its container size automatically via `ResizeObserver`. The spiral radius is computed as `min(width, height) × 0.4` to prevent overflow.

When the container width drops below 300px, zoom controls are hidden to avoid clutter.

<Canvas of={SpiralTimelineStories.SmallContainer} />

## Accessibility

- Data nodes have `role="button"`, `tabindex="0"`, and `aria-label` with the node title and formatted date
- Keyboard navigation: Enter or Space on a focused node triggers `onNodeClick`
- Tooltip appears on focus and hides on blur
- The time window slider has `role="slider"` with proper ARIA attributes
